# Ninobert Dev Portfolio

A responsive personal portfolio website built to showcase my profile, technical skills, featured projects, education, and professional background as a Computer Engineering graduate and aspiring software engineer.

## Overview

This project serves as my personal developer portfolio. It highlights my background, experience, and selected work through a clean, responsive interface built with modular front-end assets.

The site includes the following sections:

- Home / Hero
- Professional Experience
- Skills
- Projects
- Education
- Contact

## Features

- Responsive portfolio layout
- Sticky navigation header
- Modular CSS architecture
- Interactive projects slider built with JavaScript
- Layered project cards with hover-driven content reveal
- Resume and certificate download links
- Organized assets for CSS, JavaScript, images, and files

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES Modules)

Getting Started

To run the project locally:

1.Clone the repository:

```bash
git clone https://github.com/SuelloNB/Ninobert-dev-portfolio.git
```
2.Open the project folder:

```bash
cd Ninobert-dev-portfolio
```

3.Open index.html in your browser.

For a smoother development workflow, you can use the VS Code Live Server extension.

Customization

You can personalize this portfolio by updating:

Text content in index.html
Section styles inside assets/css/
Project slider behavior in assets/js/pages/projects.js
Shared initialization logic in assets/js/main.js
Images and downloadable files inside assets/images/ and assets/files/
Purpose

This portfolio was built to present my background and projects in a professional, organized, and accessible format for recruiters, hiring teams, collaborators, and potential clients.
